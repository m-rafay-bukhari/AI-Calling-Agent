📞 AI Call Agent Script Generator

An AI-powered web application designed to help businesses and individuals generate professional, high-converting call scripts in seconds. Whether it's for Real Estate sales in Urdu or a Customer Support follow-up in English, this tool automates the script-writing process using advanced LLMs.

---

## 🚀 The Problem & Solution
**Why?** Writing effective call scripts is time-consuming and requires specific sales psychology. Small business owners often struggle to find the right words to convert leads.
**What?** This app allows users to input basic business details and instantly receive a structured script.
**How?** By leveraging AI (OpenAI/Gemini) to process user inputs (Business Type, Language, Purpose) and output a tailored conversation flow.
